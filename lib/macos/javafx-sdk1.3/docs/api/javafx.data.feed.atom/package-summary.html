Package javafx.data.feed.atom

Atom Base class for the elements of an Atom feed.
AtomTask A Task that polls Atom feeds.
Category An element which represents information about a category associated with an Entry, or Feed, .
Content A Content describes the content of the entry.
Date Represents a date element in the Atom feed.
Entry An element which represents an individual entry, acting as a container for metadata and data associated with the entry.
Factory Factory class for creating new Atom elements.
Feed Feed element is top level element of an Atom Feed Document as defined in the Atom Syndication Format
Generator An element which indentifies the software agent used to generate the Feed, .
Id An element which is universally unique and a permanent identifier for an Entry, or Feed, .
Link A Link element defines a reference from an entry or feed to a Web resource.
Person A Person is an element that describes a person.
