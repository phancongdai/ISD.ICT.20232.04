The subclass of PathElement which closes the current path.

CubicCurve

The CubiCurve class defines a cubic Bézier parametric curve segment in (x,y) coordinate space.

CubicCurveTo Creates a curved path element, defined by three new points, by drawing a Cubic Bézier curve that intersects both the current coordinates and the specified coordinates (x,y) , using the specified points (controlX1,controlY1) and (controlX2,controlY2) as Bézier control points.
